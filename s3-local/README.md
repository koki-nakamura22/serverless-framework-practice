# s3-local

## Plugins

### Install
```sh
sls plugin install -n serverless-s3-local
sls plugin install -n serverless-offline
```

### Usage

#### s3-local
Make a directory for local bucket
```sh
mkdir /tmp/local-bucket
```

#### serverless-offline
Launch with s3-local
```sh
sls offline start
```
